Reflection:

The Yearbook activity introduced me to the basic elements of web page design, html, css, and Bootstrap. I had a little bit of html experience from middle school computers class and learned some css on my own before this summer camp. However, I did not have any experience with Bootstrap. I relied a lot on Stack Overflow and the Bootstrap website to learn Bootstrap syntax other than rows and columns. Bootstrap helped me with the layout of my page and saved me a lot of time on css. My main difficulty were overlays on my page. Although, I did have some basic background on css, I did not know how to do more complicated things in css like overlays. I referred to Stack Overflow and found some code that I adapted to my code. Another smaller difficulty I faced was sizing each component to make sure it looked like the original yearbook page. I addressed this by using trial and error with css until the sizing and location looked the same as the original.
