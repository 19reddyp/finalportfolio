Reflection:

For the final project, we were tasked with creating a full stack application for box inventory. As a group, we assigned each member a different thing to focus on. I was assigned to styling. This included styling in both HTML and CSS. The biggest problem I faced was waiting for the views to be styled. We should have structured development better as the way we laid it out, I had to wait for the views to be completed to style them. This caused lots of last-minute styling that could have looked much better. While styling this page, I learned a lot of new styling syntax. I learned how to make a photo slideshow and a sticky navbar. This will be helpful in styling pages in the future. This project also gave me more experience with source control and how to solve conflicts.
